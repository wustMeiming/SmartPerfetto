# 历史档案

`docs/archive/` 保存历史计划、spike、阶段决策、review 记录和已经落地的开发任务文档。这些文档有背景价值，但不代表当前推荐实现。

当前权威入口：

- [文档中心](../README.md)
- [架构总览](../architecture/overview.md)
- [agentv3 运行时](../architecture/agent-runtime.md)
- [API 参考](../reference/api.md)
- [测试规则](../../.claude/rules/testing.md)

## 子目录

| 目录 | 内容 |
|---|---|
| `context-engineering/` | agentv3 context engineering v1/v2.1 方案、SDK spike、阶段决策 |
| `architecture-reviews/` | 架构 review、bug 审查、spike 和旧优化计划 |
| `development/` | 已完成或过期的具体开发任务计划、TODO 和验收记录 |
| `features/` | feature 开发计划、review round、验收证据和历史 release note |
| `skills/` | Skill 系统历史审计和扩展计划 |
| `superpowers/` | Superpowers / Spark / case knowledge 相关历史规格与施工计划 |
| `local-scratch/` | 本机忽略的个人施工记录；该目录由 `.gitignore` 排除，不作为公开文档 |
